Backend Engine

Sistem ini dibangun secara kustom menggunakan Laravel Framework untuk mengelola alur kerja perusahaan yang kompleks. Fokus utama proyek ini adalah integrasi data antara MySQL database dengan sistem pelaporan otomatis dan mesin notifikasi email untuk meningkatkan efisiensi operasional di PT Semen Indonesia (Persero) Tbk.

Core Functionalities

  • Automated PDF Reporting: Menghasilkan dokumen laporan resmi secara dinamis langsung dari database.
  • Email Notification Engine: Sistem pengiriman email otomatis (SMTP) untuk alert dan persetujuan.
  • Relational Database: Arsitektur MySQL yang dioptimasi untuk integritas data enterprise.
  • Eloquent ORM Management: Pengelolaan data yang efisien dan aman dari SQL Injection.
  • Custom UI Components: Antarmuka unik yang dibangun dengan CSS murni tanpa library pihak ketiga.
